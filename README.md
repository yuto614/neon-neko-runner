# NEON NEKO RUNNER

ネオン×サイバーパンクな世界観で猫を操作する、ブラウザで遊べるシンプルな障害物回避アクションゲームです。

## ゲーム概要

* 自動で走り続ける猫を操作し、迫ってくる障害物をジャンプで回避してスコアを伸ばします。
* 道中に出現する魚を集めるとボーナススコアが入ります（レア魚はさらに高得点）。
* プレイを続けるとレベルが上がり、背景や障害物の種類が変化していきます。
* ニアミス・コンボ・実績解除など、繰り返し遊びたくなる要素を搭載しています。

## 操作方法

| 操作 | 内容 |
| --- | --- |
| スペースキー / クリック / タップ | ジャンプ（空中で1回だけ追加ジャンプ可能） |
| スペースキー / クリック / タップ | タイトル・ゲームオーバー画面でスタート |
| P キー | 一時停止 / 再開 |
| A キー | タイトル画面から実績一覧の表示 / 戻る |
| ESC キー | 実績一覧から戻る |

PC・モバイルの両方に対応しています。

## 主な機能

* ネオン×サイバーパンクな世界観（星空・ビル群・ホログラム広告など）
* レベルシステムによる背景・難易度の変化
* ニアミス判定とコンボボーナス
* 通常魚・レア魚の収集とスコアボーナス
* 実績システム（5種類、タイトル画面から一覧確認可能）
* 障害物バリエーション（複数種類、レベルに応じて出現）
* ダブルジャンプ
* 一時停止機能
* ハイスコア・収集データのローカル保存（localStorage）
